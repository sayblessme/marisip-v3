# MariSIP Landing Page

Лендинг для компании МариСип — строительство SIP-домов под ключ.

## Структура проекта

```
marisip-new/
├── index.html              # Основная страница
├── assets/
│   ├── css/
│   │   └── style.css       # Стили
│   ├── js/
│   │   └── main.js         # JavaScript логика
│   ├── img/                # Изображения
│   └── pdf/
│       └── catalog.pdf     # Каталог (заглушка)
└── README.md               # Документация
```

## Функционал

- **Лендинг** — одностраничный сайт с якорной навигацией
- **Квиз** — 7-шаговый опросник с прогрессом и валидацией
- **Формы** — отправка заявок через Telegram Bot API
- **Модальные окна** — консультация и расчёт стоимости
- **Слайдер отзывов** — с поддержкой touch-свайпов
- **Мобильное меню** — адаптивное бургер-меню
- **Яндекс отзывы** — виджет с отзывами на Яндекс Картах

## Настройка Telegram бота

### 1. Создание бота

1. Найдите в Telegram бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный **токен** (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Получение Chat ID

**Вариант A — личные сообщения:**
1. Отправьте любое сообщение вашему боту
2. Откройте в браузере: `https://api.telegram.org/bot<TOKEN>/getUpdates`
3. Найдите `"chat":{"id":123456789}` — это ваш Chat ID

**Вариант B — группа:**
1. Добавьте бота в группу
2. Отправьте сообщение в группу
3. Откройте: `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Chat ID группы будет отрицательным: `-123456789`

### 3. Настройка в коде

Откройте файл `assets/js/main.js` и замените значения в начале файла:

```javascript
const TG_BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
const TG_CHAT_ID = 'ВАШ_CHAT_ID';
```

## Развёртывание

### Локально

Просто откройте `index.html` в браузере. Для полной функциональности рекомендуется использовать локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```

### На хостинге

1. Загрузите все файлы на хостинг через FTP
2. Убедитесь, что `index.html` находится в корне сайта
3. Настройте Telegram токен в `main.js`
4. Замените `assets/pdf/catalog.pdf` на реальный каталог
